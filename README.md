## Xanthos water management

This directory contains scripts required to rerun the experiments presented in the publication. The required inputs data can be obtained from 
https://doi.org/10.5281/zenodo.7557403

For step-by-step instructions to set up the environment and run the model with existing calibrated parameters, see, open [docs/run_with_calibrated_parameters.ipynb](docs/run_with_calibrated_parameters.ipynb) in JupyterLab.
